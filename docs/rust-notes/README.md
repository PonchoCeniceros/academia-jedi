# Rust Notes

Presentación interactiva creada con [Slidev](https://github.com/slidevjs/slidev) que recopila notas de estudio sobre el lenguaje de programación Rust. Está diseñada como una guía visual para repasar conceptos fundamentales y avanzados del lenguaje, combinando teoría con ejercicios prácticos.

## Contenido

- **A — Fundamentos**
  - Tokens (keywords, identificadores, literales, operadores, puntuaciones, delimitadores)
  - Estructuras de Control (`if`, `match`, `if let`, `for`, `while`, `while let`, `loop`)
  - Tipos de Dato (numéricos, texto, booleanos, colecciones, personalizados)
  - Tipos de Memoria (Stack vs Heap, Copy vs Move)

- **B — Ownership, Borrowing y Lifetimes**
  - Ownership
  - Borrowing y Lifetimes
- **Ejercicios** — 56 ejercicios intercalados con respuestas

## Iniciar

```bash
pnpm install
pnpm run dev
```

Visita <http://localhost:3030>.
